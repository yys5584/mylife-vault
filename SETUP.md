# SETUP — 인터뷰 가이드 (수동 / Codex / 모든 LLM)

> Claude Code에서는 `/mylife-setup` 슬래시 커맨드 사용. 그 외 환경에서는 이 문서 직접 따르기.
>
> AI에게 이 파일을 보여주고 "이 가이드대로 인터뷰 진행해줘"라고 말하면 자동 진행됨.

---

## 인터뷰 원칙

- **분할 진행 권장**: 한 번에 30분 이상 깊은 자기성찰 못 함. 나눠서.

### 페이스 3가지 (답변 깊이에 따라)

| 페이스 | 답변 스타일 | 필수만 (1-5) | 전체 (선택 포함) |
|---|---|---|---|
| **빠르게** | 한 줄 단답 | ~30분 | ~65분 |
| **보통** | 생각하며 답 | ~50분 | ~105분 |
| **깊이** | 사례·검증·후속 질문 | ~90분 | ~180분 |

→ 처음 채울 때는 **빠르게**로 일단 다 채우고, 나중에 분기 갱신 때 깊이 보강 추천.

### 분할 가이드

- **1차 (10-30분)**: 섹션 0(기본) + 1(사주·별자리 자동) + 4(시스템) — 즉시 가치
- **2차 (15-50분)**: 섹션 2(자기) + 3(철학) — 깊은 작업
- **3차 (10-30분)**: 섹션 5(컴파스·로드맵) 필수 + 6·7 해당 시
- **4차 (15-60분, 선택)**: 섹션 8(연애)·9(투자)·10(직업) — 관심 도메인만
- 한 번 멈춰도 OK. 채워진 부분은 그대로 활용 가능.
- 모르는 항목은 "모름"으로 두고 나중에 갱신.
- 답변은 짧게. AI가 후속 질문으로 살 붙임.
- 모든 답은 **사용자 본인의 언어**로. AI가 미사여구 추가 ❌.

### AI 후속 질문 패턴 (가이드)
사용자 답이 한 줄이면 AI가:
1. 구체 사례 요청 ("최근 그 패턴이 발현된 사건 1개?")
2. 반대편 확인 ("그럼 안 그런 영역도 있나?")
3. 검증 ("이 답을 1년 전에도 같이 했을까?")
한 번에 1개 후속 질문만. 답 받으면 다음.

## 섹션 1 — 사주 + 별자리 계산 (자동, 10분)

생년월일·시간·장소가 모이면:

**Python 있는 환경:**
```bash
pip install -r scripts/requirements.txt

# 사주 (동양)
python scripts/calc_saju.py \
  --date 1990-03-15 \
  --time 14:30 \
  --gender male \
  --place "Seoul" \
  --true-solar

# 별자리 (서양)
python scripts/calc_zodiac.py \
  --date 1990-03-15 \
  --time 14:30 \
  --place "Seoul"
```

→ `templates/saju.md` + `templates/zodiac.md` 자동 생성.

**⚠️ 도시가 인식 안 되면**:
인식 가능: 서울/부산/인천/대구/대전/광주/수원/제주/도쿄/오사카/베이징/상하이 + 별자리는 추가 (뉴욕/LA/런던/파리/시드니).
**그 외 도시면 위경도 직접 입력**:
```bash
# 예: lat=XX.XXXX, lon=XXX.XXXX (구글 지도에서 출생지 좌표 복사)
python scripts/calc_saju.py --date 1990-03-15 --time 14:30 --gender male \
  --longitude XXX.XXXX --true-solar
python scripts/calc_zodiac.py --date 1990-03-15 --time 14:30 \
  --lat XX.XXXX --lon XXX.XXXX --tz 9
```
위경도는 구글 지도에서 출생지 검색 → 우클릭 → 좌표 복사.

**Python 없는 환경:**
1. **사주**: 만세력 사이트 (원광대 만세력) → 8자 조회 → `saju.md` 표에 직접 입력
2. **별자리**: astro.com (Free Horoscopes → Birth Chart) → 빅3 + 행성 → `zodiac.md` 표에 직접 입력
3. AI에게 "내 사주·별자리 해석해줘" → 각 .md의 해석 섹션 채우기

**왜 둘 다?** 사주(동양)와 별자리(서양)는 다른 좌표계. 공통으로 강한 영역 = 본질이 강하게 드러남. 충돌 영역 = 본인 안의 다층성.

## 섹션 2 — 자기 진단 (30분)

`templates/self_profile.md` 채움. **가장 깊은 작업.** 막히면 다음으로 넘어가도 됨.

질문 순서를 강점 → 약점 → 패턴 → 환경 → 위험 신호로 (가벼운 것부터):

```
1. 본인 강점 5-7개. 1-10점 자체 평가 + 검증 사례 (실제 일어난 일).
   막히면: "남들이 자주 칭찬하는 거 3개 + 본인이 자랑하고 싶은 거 3개"

2. 본인 약점 3-5개. 1-10점 + 작동 방식 (어떤 상황에서 발현).
   막히면: "후회 가장 많은 패턴 3개"

3. 본인의 *반복 루프*는? (트리거 → 즉시 행동 → 결과)
   (예: "불안 → 새 사이드 시작 → 본업 더 흔들림")
   막히면: "최근 한 달 후회한 사건 3개 떠올려서 공통점"

4. 영역별 발현 (위 루프가 어떻게 다르게 나타나는지):
   - 일에서 트리거·행동?
   - 관계에서 트리거·행동?
   - 돈에서 트리거·행동?
   - 몸에서 트리거·행동?

5. 가정환경·성장 배경 — 현재의 본인을 만든 핵심 사건 3-5개.
   (가족 구도·결정적 사건·가치관 형성 시점·트라우마 등 무엇이든)

6. 신경·건강 진단 (ADHD·우울·불안 등 있으면)?
   - 약물 복용 중이면?
   - 수면 패턴?
   - 운동 빈도?

7. 포장(겉) vs 실물(속) — 남에게 보이는 vs 본인이 아는 실제. 각 3개.
   막히면: "비판 받았을 때 가장 아팠던 말" → 그게 실물 / 그 반대가 포장

8. 본인 한 줄 요약 (1-7 답 다 보고 마지막에)
   (예: "에너지 많은데 방향이 없음" / "분석은 9짜리, 행동은 3짜리")

9. 본인이 무너지기 직전 위험 신호 3개?
   (예: 잠 4시간 연속, 새 사이드 시작 충동, 카페인 3잔)
   막히면: "이전에 무너졌을 때 직전 1주일에 어떤 사인이 있었나?"
```

**자동 보강**: 사주 8자 + 별자리 빅3 결과를 본인 답과 매칭해서 일치/충돌 영역 자동 코멘트.

## 섹션 3 — 인생 철학 (20분)

`templates/philosophy.md` 채움.

```
1. WHY — 가치 3개 (다 빼앗겨도 *마지막에 사수*할 것).
   reframe 막히면: "감옥 들어가도 끝까지 안 포기할 것 3개"
   (예: 자유·탐험·동반 / 안전·성장·가족 / 영향·정직·자유)

2. WHAT — 핵심 활동 3개? (정체성을 *실제로* 어떻게 표현하는가)
   막히면: "지난 1년에 가장 많은 시간을 쓴 활동 3개"

3. WHAT — 정체성 한 문장 (Q1·Q2 답 다 보고 마지막에):
   "나는 [무엇을] 하는 [어떤] 사람이다."
   매월 자문: "명예·지위 없어도 나는 이걸 할 건가?"

4. HOW — 작업 모델? (혼자 / AI 활용 / 팀 / 외주 비중 — 백분율로)

5. HOW — 비협상 베이스라인 3개?
   (예: 매일 4시간 집중 / 주 4일 노동 / 운동 주 4회)

6. HOW — 운영 원칙 3-5개?
   (예: "단순함 = 도덕" / "위기 = 자산 전환" / "선제적 공유")
```

**자동 보강**: saju.md의 일간(日干) 기질 + zodiac.md의 태양궁 trait를 한 줄 요약해서 philosophy의 *사주 ↔ 철학* 섹션에 자동 채움.
